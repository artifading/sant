Privacy Policy

Last updated: June 2026

This is a personal dashboard application built for private use by Miguel Santos. It connects to WHOOP to display personal health and recovery data.

Data Collection

This application does not collect, store, or share any personal data with third parties. All data retrieved from WHOOP via OAuth is displayed only to the authenticated user and is stored locally in the user's own browser (localStorage) or in the user's own Supabase project.

WHOOP Data

WHOOP recovery, sleep, strain, HRV, and biometric data is accessed via the official WHOOP API using OAuth 2.0. This data is only used to display metrics to the account owner. No WHOOP data is transmitted to any server other than the user's own backend proxy.

Third-Party Services

This application may use Supabase (for personal data sync across devices) and Vercel (for hosting). No health data is shared with these services beyond what is necessary for the user's own sync.
